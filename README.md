# Contador Interativo

Este projeto foi desenvolvido como um exercício de fixação para praticar a manipulação do **DOM (Document Object Model)** em JavaScript. O objetivo principal foi consolidar conceitos de eventos, funções e atualização dinâmica de elementos HTML antes de uma avaliação acadêmica.

## O Projeto

A aplicação consiste em um contador simples onde o usuário pode incrementar, decrementar e resetar o valor de uma variável. Para tornar o aprendizado mais divertido, personalizei a interface com:
- **Comic Sans**: Para um visual irreverente e amigável.
- **Made in Heaven**: O botão de resetar é uma referência ao Stand de *JoJo's Bizarre Adventure*, que reseta o universo (e aqui, o contador).

## Tecnologias Utilizadas

- **HTML5**: Estruturação da página.
- **CSS3**: Estilização e layout com Flexbox.
- **JavaScript (ES6+)**: Lógica do contador e manipulação de elementos em tempo real.

## Conceitos Praticados

- [x] Seleção de elementos com `document.getElementById`.
- [x] Escuta e manipulação de eventos (`onclick`).
- [x] Atualização de conteúdo dinâmico via `innerText`.
- [x] Lógica de programação (incremento, decremento e atribuição).
